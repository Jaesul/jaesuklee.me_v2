hello.
I'm a developer and UX designer

My name is Jae, I’m a web developer and UX designer from Seattle. This website is for you to look at and determine that I’m worthy of a job. I’m really good with people, my mom says I’m the best actually, so it was only fitting that I pursue a career that focuses on meeting the needs of people. That’s the UX part of my skills, finding what users want and creating beautiful, simple designs. I’m also a computer wiz, often turning on and off devices to magically fix them, because that’s what someone studying computer science does. The development skills come from my affinity to technical problem solving. It’s great, having the ability to design and execute all at the tip of my fingers. Please hire me.

CSS: Why can't you just do what I want?

The most recent coding language that I've added to my toolbelt. JavaScript is my favorite language its simple to use (in my perspective), and is the primary language for the web. It’s funny because you always start out trying to optimize and learn a language that’s the perfect fit for you, but for web that decision is already made for you. Of course, there’s TypeScript and other *ahem* no name coding languages for the web (kidding) but JavaScript is the most popular and with the most support. 9.5/10 Would code with again.

HTML: *pushes up glasses* Actually it's not a 'coding' language'

It’s pretty simple, I have to admit HTML isn’t bad at all. If only I could remember how to connect style sheets or JavaScript code. It’s always funny to see your browser history fill up with ‘how to link css html’. HTML is great, it’s not difficult to understand, and easy to see structure using tabs. 8/10 Would code again because of ease of use.
